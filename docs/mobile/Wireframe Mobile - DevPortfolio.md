# Wireframe Mobile - DevPortfolio

Este documento descreve o wireframe mobile para o portfólio do desenvolvedor, adaptado do layout desktop fornecido. A estrutura visa otimizar a experiência do usuário em dispositivos móveis, priorizando a legibilidade e a navegação intuitiva.

## Estrutura Geral

A navegação será simplificada, com um menu hambúrguer para acessar as seções principais. O conteúdo será empilhado verticalmente, com os elementos mais importantes aparecendo primeiro.

## Seções do Wireframe

### 1. Cabeçalho (Header)

- **Logo:** "DevPortfolio" no canto superior esquerdo.
- **Menu Hambúrguer:** Ícone no canto superior direito para expandir a navegação (Home, About, Projects, Skills, Contact).

### 2. Seção Inicial (Hero Section)

- **Saudação:** "Hi, I'm Alex"
- **Título:** "Developer Portfolio"
- **Descrição:** "I build modern, responsive, and scalable web applications."
- **Botões de Ação:** "View Projects" e "Contact Me" (empilhados verticalmente ou em linha, dependendo do espaço).
- **Imagem/Ilustração:** A imagem do monitor com `< />` será removida ou adaptada para um elemento visual menor e mais discreto, ou colocada abaixo dos botões de ação.

### 3. Sobre Mim (About Me)

- **Título:** "About Me"
- **Imagem de Perfil:** Placeholder circular.
- **Texto:** "I'm a passionate full-stack developer..."
- **Botão:** "Download Resume"

### 4. Habilidades (Skills)

- **Título:** "Skills"
- **Ícones de Habilidades:** Ícones como React, Node.js, Python, JavaScript, HTML, CSS, dispostos em um grid responsivo (2 ou 3 colunas).

### 5. Projetos (Projects)

- **Título:** "Projects"
- **Cards de Projeto:** Cada projeto (Project One, Project Two, Project Three, Project Four) será exibido em um card individual, empilhado verticalmente.
  - **Placeholder de Imagem:** Para cada projeto.
  - **Título do Projeto:** Ex: "Project One"
  - **Descrição do Projeto:** Ex: "A responsive web application..."
  - **Botão:** "View Project"

### 6. Contato (Contact Me)

- **Título:** "Contact Me"
- **Formulário de Contato:** Campos "Name", "Email", "Message" (empilhados verticalmente).
- **Botão:** "Send Message"
- **Seção "Let's connect!":** Texto e ícones de redes sociais (GitHub, LinkedIn, Twitter, Email) dispostos horizontalmente ou empilhados.

### 7. Rodapé (Footer)

- **Direitos Autorais:** "© 2024 Alex Developer. All rights reserved."
