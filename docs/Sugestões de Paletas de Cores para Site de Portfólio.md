## 1. Paleta: "Midnight Tech" (Dark Mode Moderno)
Esta é a escolha mais popular para desenvolvedores atualmente. Transmite modernidade, foco em código e é confortável para os olhos.

| Elemento | Cor (Hex) | Amostra | Uso Recomendado |
| :--- | :--- | :--- | :--- |
| **Fundo Principal** | `#0F172A` | 🌑 | Fundo de todas as seções. |
| **Fundo Secundário** | `#1E293B` | 🌚 | Cards de projetos e seções de destaque. |
| **Destaque (Primary)** | `#38BDF8` | 💎 | Botões CTA, ícones de habilidades e links. |
| **Texto Principal** | `#F8FAFC` | ⚪ | Títulos e parágrafos importantes. |
| **Texto Secundário** | `#94A3B8` | 🔘 | Descrições menores e subtítulos. |

## 2. Paleta: "Clean Professional" (Light Mode Minimalista)
Ideal para uma imagem mais "Enterprise" ou corporativa. Transmite clareza, organização e profissionalismo direto.

| Elemento | Cor (Hex) | Amostra | Uso Recomendado |
| :--- | :--- | :--- | :--- |
| **Fundo Principal** | `#FFFFFF` | ⚪ | Fundo limpo e arejado. |
| **Fundo Secundário** | `#F3F4F6` | 🥈 | Divisão de seções e backgrounds de inputs. |
| **Destaque (Primary)** | `#4F46E5` | 🍇 | Identidade visual forte em botões e títulos. |
| **Texto Principal** | `#111827` | ⚫ | Máxima legibilidade para conteúdo. |
| **Destaque Suave** | `#E0E7FF` | 🧊 | Background de badges (tags de tecnologias). |

## Dicas de Aplicação no seu Wireframe:

1.  **Botão de Download CV:** Use a cor de **Destaque (Primary)** com texto em alto contraste.
2.  **Habilidades (Tech Stack):** Use a cor de **Fundo Secundário** para os "badges" e a cor de **Destaque** para os ícones.
3.  **Cards de Projetos:** Mantenha uma borda sutil ou sombra usando a cor de **Texto Secundário** com baixa opacidade para dar profundidade.
4.  **Contraste:** Sempre verifique se a cor do texto sobre o fundo escolhido possui leitura fácil (WCAG Accessibility).
